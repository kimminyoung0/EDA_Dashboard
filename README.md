# EDA_Dashboard
데이터 분석을 모듈화시켜 간편하고 일관성 있는 분석을 위해 구현했습니다.

------
## 🛠️ 기술 스택
- **언어**: Python
- **라이브러리**: Scikit-learn, Matplotlib, Seaborn
- **데이터 분석 & 처리**: Pandas, NumPy
- **배포**: Streamlit
